DIVs

The DIV tag on its own doesn't do much.

DIV tag definition

  • The DIV tag has no default styling
  • DIV tag defines content areas
  • HTML is not funny

DIV tag usage

Assigning an id tag allows us to style the tag (and all its nested tags) directly through CSS.

To give that id styling it must be named as a selector, along with its declaration block, and preceded by using a pound sign (hashtag symbol).

ID usage

The id allows us to identity tags so that they may specifically be addressed within the CSS file.
